#How to run the code for Baseline LSTM model:

## Requirements
1. Python 3.8+ installed
2. PyTorch installed (tested with torch 2.1+)
3. IOB2 data files are available in the same directory as the scripts:
- en_ewt-ud-train.iob2
- en_ewt-ud-dev.iob2
- en_ewt-ud-test-masked.iob2

## Scripts
1. **Baseline_LSTM.py**
- Trains a bidirectional LSTM on the training set.
- Predicts on the test set and save predictions---> Output file:`test_predictions_LSTM.iob2`

2. **Dev_predictions_LSTM.py**
- Trains the same LSTM model.
- Saves predictions on the development set ----> Output file: `dev_predictions_LSTM.iob2`

## Running the Code
### To train on en_ewt-ud-train.iob2 and predict on en_ewt-ud-test-masked.iob2: python Baseline_LSTM.py
### To train on en_ewt-ud-train.iob2 and predict on en_ewt-ud-dev.iob2: python Dev_predictions_LSTM.py

## Notes
- Training loss is printed each epoch.  
- Output files are in CoNLL IOB2 format.
